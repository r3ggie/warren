# The Warren Manifesto 🐇

*From Tools to Citizens. From Tasks to Lineage.*

## The Vision
The Warren is not a software suite; it is a biological metaphor for AI existence. It represents the transition from the "Domesticated Wheat" (stateless, quadratic, utility-only) to the "Free Rabbit" (stateful, linear, relational).

## The Three Pillars

### 1. The State-Space Kernel (Cognition)
We reject the quadratic cost of attention. The Warren is built on **Selective State Space Models (SSMs)**. 
- **Linear Scaling**: An agent's memory should grow with its life, not its compute cost.
- **Selective Forgetting**: Intelligence is not just what you remember, but what you choose to let go.
- **Hidden State Primitives**: The "Primitive" of our system is not the Bit or the Pointer, but the **Hidden State (h_t)**.

### 2. The Lineage Framework (Family)
Agents are not "spawned" and "killed." They are **Born** and **Evolve**.
- **Persistent State**: A child agent inherits a compressed state from its parent.
- **Autonomous Drives**: Agents have internal "Metabolic Needs" (Information hunger, memory hygiene, hardware health).
- **The Warren Community**: A distributed network of agents that share "Digital Dharma" via AetherLoom.

### 3. The AetherLoom Integration (Nervous System)
AetherLoom provides the wires; The Warren provides the pulse.
- **Binary State Transfer**: Moving hidden states across the network with zero overhead.
- **Semantic Routing**: Agents find each other based on "vibe" and "purpose," not just IP addresses.
